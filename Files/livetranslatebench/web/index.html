Run the benchmark, regenerate the leaderboard, and redeploy. The page reads leaderboard.json from this folder.

ltbench run --system gemini-3.5-live-translate \
  --manifest data/fleurs_es_en.jsonl --snr clean,20,10,5 \
  --noise data/noise/babble_16k.wav --out results/gemini.jsonl

ltbench leaderboard --results results   # writes web/leaderboard.json
git commit -am "update leaderboard" && git push   # Vercel redeploys

Lower is better: PAL, TTFA, finish lag, RTF, WER. Higher is better: BLEU, chrF, COMET. All latency measured on the shared real-time harness clock. Numbers shown are only those produced by published runs.
